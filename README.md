# 📘 Utility Hub

**Utility Hub** — консольное приложение на C#, включающее набор полезных утилит: калькулятор, конвертер температур, анализатор чисел и генератор паролей. Приложение удобно для быстрого использования прямо из консоли.

---

## 📌 Возможности

### 1️⃣ Калькулятор

* Сложение, вычитание, умножение, деление
* Защита от деления на ноль
* Удобное меню с выбором операций

### 2️⃣ Конвертер температур

* Цельсий → Фаренгейт
* Фаренгейт → Цельсий
* Проверка корректности ввода

### 3️⃣ Статистика чисел

* Ввод чисел через пробел
* Определение:

  * Минимального значения
  * Максимального значения
  * Суммы
  * Среднего
  * Количества положительных
  * Количества отрицательных

### 4️⃣ Генератор паролей

* Выбор длины пароля
* Заглавные и строчные буквы
* Цифры и спецсимволы
* Безопасная генерация случайного пароля

## 🛠 Установка и запуск

### 1. Установка .NET SDK

Убедитесь, что установлен .NET 6 или выше:

```bash
dotnet --version
```

Если SDK не установлен, скачать с официального сайта:

[https://dotnet.microsoft.com/download](https://dotnet.microsoft.com/download)

---

### 2. Скачивание проекта

**Вариант A: Скачать ZIP**

1. На GitHub нажать **Code → Download ZIP**
2. Распаковать архив

**Вариант B: Клонировать репозиторий**

```bash
git clone https://github.com/username/UtilityHub.git
```

---

### 3. Открытие проекта

Открыть в любой IDE:

* Visual Studio
* Rider
* Visual Studio Code
* Или работать через консоль

---

### 4. Запуск проекта

**Через IDE:**
Нажать кнопку **Start** (зелёная кнопка).

**Через консоль:**

```bash
cd UtilityHub
dotnet run
```
